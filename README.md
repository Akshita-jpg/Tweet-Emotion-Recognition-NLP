## 📌 Project Overview

This project builds a **Tweet Emotion Classification system** using Natural Language Processing techniques.

The model predicts the emotion expressed in a tweet such as:

* Joy 😊
* Anger 😡
* Sadness 😢
* Fear 😨
* Love ❤️
* Surprise 😲

This is a **multi-class text classification problem**.

---

## 🛠 Complete ML Pipeline Implemented

### 1️⃣ Data Loading

* Loaded dataset of tweets
* Checked dataset shape
* Verified class distribution
* Handled missing values

### 2️⃣ Text Preprocessing

* Lowercasing
* Removing punctuation
* Removing stopwords
* Tokenization
* Lemmatization / Stemming

### 3️⃣ Feature Engineering

* Text vectorization using:

  * TF-IDF / CountVectorizer
    OR
  * Tokenizer + Padding (for Deep Learning)

### 4️⃣ Model Training

Models experimented:

* Logistic Regression
* Naive Bayes
* Neural Network / LSTM (if used)

### 5️⃣ Model Evaluation

* Accuracy
* Confusion Matrix
* Classification Report
* Precision, Recall, F1-score
